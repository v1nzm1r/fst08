GUI vs CLI
Graphical User Interface (GUI) - graphical buttons/icons.
Command Line Interface (CLI) - text-based commands.

Make a folder (Make a directory)
mkdir 'Name of folder'

Remove/Delete a folder
rmdir 'name of folder'

Print Working directory
pwd

List commands - Lists all files and folders in the current directry 
ls 

Change Directory
cd 'directory/folder name'

Go back one directory
cd ..

Create a filename
touch 'filename.extension'

Remove a file 
rm 'filename.extension'

Day 2

U - Untracked
A - Added
M - Modified

- 3 Stage Architecture
1. Working Area
2. Staging Area
3. Commit Stage

- Check the status of all the files and folders within project folders
git status

- Add all the files that have been modified
git add .
git add "filename.ext"

- Create a commit 
git commit -m "Description."

- Check the history of the changes.
git log
git log --oneline

Branching and Merging

- List all the available branches
git branch -a

- Create a new branch
git branch [branch-name]

- Switch to a different branch
git switch [branch-name]
git checkout [branch-name]
